# Meu Primeiro Repositório

Bem-vindo ao meu primeiro repositório! <br> 
Reúne diversos exercícios e aplicações práticas desenvolvidas em **HTML5**, focando na lógica de programação e estruturação web.

---

## 📂 O que tem aqui?

Este repositório contém mini-projetos independentes:

* **Calculadora:** Uma ferramenta para realizar operações matemáticas básicas.
* **Conversão:** Aplicativo para conversão de valores ou unidades.
* **Loja:** Estrutura inicial de uma interface de e-commerce.
* **Mecânica/Estação:** Exercícios temáticos de organização de conteúdo.
* **Classificação:** Lógica aplicada para categorizar dados.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação das páginas.
* **CSS3:** Estilização e layout.
* **JavaScript:** Lógica das calculadoras e conversores.

---

## 🚀 Como visualizar os projetos

Como é um repositório de arquivos estáticos, você pode:

1.  Fazer o download ou clonar este repositório.
2.  Abrir o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox).
